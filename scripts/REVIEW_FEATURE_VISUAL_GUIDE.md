### 5. Color Scheme

- **Stars**: Yellow (#FBBF24)

- **Info**: Blue (#3B82F6)

### 6. Responsive Behavior

#### Desktop (>1024px)

- Rating summary: 2 columns (average + distribution)
- Review cards: Full width with padding
- Form: Centered with max-width

#### Tablet (768px - 1024px)

- Rating summary: 2 columns (stacked on smaller tablets)
- Review cards: Full width
- Form: Full width with padding

#### Mobile (<768px)

- Rating summary: Single column (stacked)
- Review cards: Full width, reduced padding
- Form: Full width, adjusted spacing
- Star ratings: Slightly smaller

## User Interactions

### 1. Viewing Reviews

- Click "Reviews" tab on product page
- Scroll through review list
- See average rating and distribution

### 2. Writing a Review

- Click "Write a Review" button
- Click stars to select rating (hover to preview)
- Type review title
- Type review comment
- Watch character counters
- Click "Submit Review"
- See success message
- Review appears in list

### 3. Form Validation

- Rating required (1-5 stars)
- Title required (3-100 chars)
- Comment required (10-1000 chars)
- Real-time character counting
- Error messages for invalid input

## Accessibility Features

- Semantic HTML structure
- ARIA labels on interactive elements
- Keyboard navigation support
- Focus indicators on form inputs
- Color contrast compliance
- Screen reader friendly
