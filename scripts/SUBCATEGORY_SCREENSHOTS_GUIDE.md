## Important Notes

### âœ… What You CAN Do:

- Create unlimited main categories
- Create unlimited subcategories under each main category
- Edit any category and change its parent
- Convert a subcategory to a main category (set parent to "None")
- Convert a main category to a subcategory (select a parent)
- Delete subcategories anytime
- Delete main categories that have NO subcategories

### âŒ What You CANNOT Do:

- Delete a main category that has subcategories
  - Error: "Cannot delete category with subcategories"
  - Solution: Delete all subcategories first
- Create circular references (a category cannot be its own parent)
- Use duplicate names or slugs

## Tips for Best Results

1. **Create Main Categories First**: Always create your top-level categories before adding subcategories

2. **Use Descriptive Names**: Make it clear what products belong in each category

   - Good: "Men > Casual Shirts"
   - Avoid: "Category 1 > Type A"

3. **Keep It Organized**: Don't create too many levels

   - Recommended: Main â†’ Sub (2 levels)
   - Avoid: Main â†’ Sub â†’ Sub-Sub (3+ levels)

4. **Test Your Structure**: Add a test product to verify the category appears correctly

5. **Use the Search**: When you have many categories, use the search box to find them quickly
